# Pt乱数ID

## 目标

Pt 的 ID 乱数目标，是在新游戏中获得指定的 **TID/SID**。

如果这组 TID/SID 对应优秀 PID，就能让后续遇到的特定 PID 变成闪光。

## 选择 SID 范围

1. 进入 **ID乱数** 页。
2. 点击 **寻找SID**。
3. 在新页面输入想要的 **TID**。
4. **最小SID** 和 **最大SID** 可以留空。
5. 点击 **寻找优秀SID**。

这会进入优秀 SID 检索页。

## 寻找优秀 SID

优秀 SID 页会列出不同 TSV 能够遇到的优秀 PID。

建议：

- 排序使用 **性格最多**。
- **最低个体值** 表示：用来判定“优秀个体”的最低个体值。

点击 **检索优秀SID** 后，查看每张结果卡片。

每张卡片显示的是：当前 TSV 下可以闪光的优秀 PID 组合。

找到心仪的卡片后，点击它。

app 会自动返回上一页，并填充对应的 SID 范围。

## 检索实际可达 SID

回到 SID 检索页后：

1. 输入 **年份**。
2. 设置 **最小Delay** 和 **最大Delay**。
3. 点击 **检索SID**。

这里检索的是：在当前年份与 Delay 范围内，实际能够命中的 SID。

它不是上一步理论计算出的全部优秀 SID，而是进一步筛选出可通过实机时间命中的结果。

建议：

- 年份就是你想在 DS 上创建游戏存档的年份。
- 最小Delay建议至少 **5000**。
- Delay 太小的话，新游戏创建人物通常来不及操作。
- 如果检索速度太慢，可以缩小最小Delay和最大Delay之间的范围。

## 选择时间

点击想要的 SID 结果后，app 会自动返回 ID乱数页。

然后：

1. 点击 **获取Seed时间**。
2. 输入月份和日期。
3. 选择一个心仪的时间。

这里的月份和日期，会成为游戏训练家卡片上显示的游戏开始日期。

点击时间后，app 会自动返回，并把目标 Delay 填入定时器。

## 实机操作

按照定时器提示进行：

1. 设置 DS 时间。
2. 启动游戏。
3. 快速创建人物。
4. 到达电视机画面后等待。
5. 在定时器最后一阶段倒计时结束的瞬间按 A。

按键时机是倒计时结束的瞬间，也就是提示音第 4 声开始响起的瞬间。

## 校准

进入游戏后，查看实际生成的 **TID**。

回到 app：

1. 在 **实际ID校验** 输入实际 TID。
2. 点击 **反查命中Delay**。
3. 如果没有结果，可以增大 **Delay偏差**。
4. 点击反查结果。

点击结果后，app 会自动填入定时器的 **实际命中Delay**。

确认无误后，点击 **应用校准**，定时器会更新。

## 重试

校准后重复同样流程。

如果命中 Delay 只差 **±2**，可以先不急着更新校准，多试几次。

如果稳定命中同一个 Delay，再应用校准。

不断重复后，就可以逐渐稳定命中目标 Delay，获得目标 TID/SID。
