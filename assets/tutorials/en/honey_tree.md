# Pt Honey Tree Shiny

## Core Idea

Honey Tree RNG is easiest at a tree with no NPC interference.

The species is already decided when honey is slathered on the tree. During later resets in front of that tree, each seed and each frame encounters the same species; only PID, IVs, nature, level, and similar details change.

## Confirm the Species First

1. Slather honey on the tree.
2. Wait 6 hours.
3. Trigger the tree once.
4. Confirm which Pokemon is waiting.

If it is not the target:

1. Defeat or catch it.
2. Slather honey again.
3. Wait another 6 hours.

The Pokemon can appear after 6 hours, and the honey disappears after 24 hours. Changing the DS clock does not directly advance the Honey Tree wait timer.

## Choose the Target Frame

If you have Chatot available, you can search a wider frame range.

Without an advance method, prefer:

- minimum frame: 0
- maximum frame: 0

At a tree with no NPC interference, loading the save and immediately pressing A on the tree usually gives the frame 0 Pokemon.

A good starting delay range is:

- 900 to 15000

## Capture and Calibrate

1. Find the desired result on the search page.
2. Save it or send it to calibration.
3. Follow the timer to set the DS clock, start the game, and enter the save.
4. After loading, immediately press A on the tree.
5. Catch the Pokemon.
6. Record its stats.
7. Use reverse search on the calibration page to calculate the actual hit delay.
8. Tap the result and apply calibration.

Repeat timer, encounter, catch, reverse search, and apply calibration until the target Honey Tree Pokemon is stable.
