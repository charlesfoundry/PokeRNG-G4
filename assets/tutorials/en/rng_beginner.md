# What RNG Means

RNG abuse does not mean editing the game, and it does not mean forcing a creature into the save file.

It is closer to this:

```text
Study the game's own random process,
then use the right time and the right actions
so the game naturally generates the desired result.
```

## Randomness in the Game

Many game results look random:

- Which creature appears
- Its nature
- Its IVs
- Whether it is shiny
- Its gender
- Its ability slot
- The result of an egg

However, the console does not create true randomness. It starts from a value called a `seed`, then follows fixed formulas to produce later results.

Think of it as a long queue of results that already has an order:

```text
Result 1
Result 2
Result 3
...
```

During normal play, you do not know which result you are reaching, so it feels random.

RNG abuse means calculating that queue and finding a way to reach the position you want.

## Reproducing a Moment

Suppose you opened the game on a certain day, at a certain time, and performed a sequence of actions:

```text
Set the DS clock
Start the game
Load the save
Open a menu
Walk a few steps
Press A to trigger an encounter
```

At the end, you encountered a creature.

If the game state is the same, and you can reproduce:

- The same DS time
- The same timing when entering the game
- The same action order
- The same number of actions
- The same moment of triggering the encounter

then in theory, the game reaches the same position in the same random sequence and generates an extremely similar or even identical result.

> The core idea is not to create a result. It is to reproduce a specific time and a specific path of actions.

## What the App Does

It is very hard for a player to know:

```text
Which day
Which hour, minute, and second
Which Delay
Which frame
will produce the target
```

The app calculates these conditions for you.

It tells you:

- What DS time to set
- When to enter the game
- Which frame to advance to
- When to press A or trigger the encounter

In other words:

```text
The tool does not modify the result.
It helps you find "that day, that second, and that sequence of actions."
```

## Why Do RNG Abuse?

During normal play, you can naturally find:

- Shiny creatures
- Good natures
- High IVs
- Specific genders
- Specific ability slots
- Ideal eggs
- Specific TID/SID combinations

The problem is that the odds are often very low.

If you want all of these at once:

```text
shiny + good nature + high IVs + specific target
```

waiting for pure luck can take a very long time.

RNG abuse turns:

```text
repeating attempts and hoping
```

into:

```text
calculating the target position
practicing the timing
reproducing the result reliably
```

## Why the Result Is Legal

A result obtained through RNG abuse is still generated by the game itself.

During the process:

- The save file is not edited
- No creature is injected
- Memory is not modified
- PID or IV data is not forced into place
- The game's generation rules are not bypassed

The player only controls external conditions:

- DS system time
- Timing when entering the game
- Button timing
- Number of phone calls
- Number of Chatot cries
- Timing of an encounter or egg pickup

The final result is still created by the original game program.

Therefore, its data naturally follows the game's rules:

| Data | Why it is legal |
| --- | --- |
| PID | Generated by the game's RNG |
| IVs | Generated by the game's RNG |
| Nature | Determined by PID or RNG results |
| Shiny status | Determined normally from TID/SID/PID |
| Location and level | Taken from the game's own encounter data |

## Difference from Save Editing

Save editing usually means:

```text
Writing the desired data directly into the save file,
or modifying memory so the result becomes what you want.
```

RNG abuse means:

```text
Do not edit the data.
Do not edit the save.
Use time and actions so the game naturally generates the target.
```

The key difference is:

```text
Save editing creates the target by changing data.
RNG abuse makes the game generate the target by itself.
```

## One-Sentence Summary

If normal play is drawing from a result queue without knowing where the good result is, RNG abuse is:

```text
Calculate where the target result is,
reproduce the correct time and actions,
and let the game draw that result by itself.
```
