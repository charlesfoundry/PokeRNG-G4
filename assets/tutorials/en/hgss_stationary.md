# HGSS Stationary Shiny

## Core Idea

The most reliable flow is:

1. Confirm the initial seed with phone calls.
2. Use Chatot Chatter to advance to the target frame.
3. Return to the map and press A on the stationary Pokemon.

HGSS advances 1 frame for every Professor Elm or Juggler Irwin call. Seed confirmation usually takes 6 to 7 calls, so targets after frame 10 are easier to operate.

## Preparation

1. Stand in front of the stationary Pokemon.
2. Save the game.
3. Turn off the DS.
4. Start the DS, change the system time, and start the app timer at the same time.

## Timer Operation

Follow the timer beeps:

1. End of prep stage: confirm the DS clock.
2. End of stage 1: start the game.
3. End of stage 2: enter the save file.

## After Loading

If there are no moving or turning NPCs nearby and no active roamers causing initial advances, the current frame can be treated as 0.

Open the Bag or phone interface and enter the app Seed Search page. After each call, tap the exact matching E/K/P result in the same order. Possible seeds will appear below.

If you made 6 calls on the real game, tap 6 matching E/K/P results in the app.

The count and order must match exactly, or the current frame will be calculated incorrectly.

## If You Hit the Target Seed

Tap the target seed result.

The app uses the calls already made to calibrate the current frame and shows how many Chatot cries are still needed.

Listen to all Chatot cries shown by the app, return to the map, and press A while facing the stationary Pokemon. The Pokemon will be generated on the target frame.

## If You Miss the Target Seed

Tap the seed you actually hit.

The app returns to the calibration page and fills the delay actually hit in this attempt.

The timer is updated only after you tap Apply Calibration.

If the hit delay differs from the target delay by only **±2**, you can avoid updating calibration yet and try a few more times with the current calibration first.

If the same offset appears consistently, then apply the calibration.
