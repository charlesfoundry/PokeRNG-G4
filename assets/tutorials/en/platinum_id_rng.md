# Pt ID RNG

## Goal

Pt ID RNG is used to start a new save with a target **TID/SID** pair.

If that TID/SID pair matches useful PID targets, those PIDs can become shiny later in the save.

## Choose a SID Range

1. Open the **ID RNG** page.
2. Tap **Find SID**.
3. Enter the **TID** you want.
4. Leave **Min SID** and **Max SID** blank if you do not have a range yet.
5. Tap **Excellent SID Finder**.

This opens the excellent SID search page.

## Find an Excellent SID

The excellent SID page lists strong PID groups that can be shiny for each TSV.

Recommended settings:

- Use **Most Natures** as the sort order.
- **Minimum IV** is the minimum IV used to decide whether a PID is considered excellent.

Tap **Search Excellent SID**, then inspect the result cards.

Each card shows the excellent PID targets available for that TSV.

Tap a card you like.

The app returns to the previous page and fills the corresponding SID range.

## Search Reachable SID Results

Back on the SID search page:

1. Enter the **year**.
2. Set **Min Delay** and **Max Delay**.
3. Tap **Search SID**.

This searches SID results that are actually reachable with the selected year and Delay range.

It is not the full theoretical excellent SID list from the previous step; it is the subset that can be hit by real timing.

Recommendations:

- The year is the year you want to use when creating the DS save.
- Use a minimum Delay of at least **5000**.
- Very low Delay targets are usually not practical because new-game setup takes time.
- If the search is slow, narrow the Min Delay and Max Delay range.

## Choose a Time

After tapping the SID result you want, the app returns to the ID RNG page.

Then:

1. Tap **Get Seed Time**.
2. Enter the month and day.
3. Choose a time you want to use.

The month and day become the start date shown on the Trainer Card.

After you tap a time, the app returns and fills the target Delay into the timer.

## Console Steps

Follow the timer:

1. Set the DS time.
2. Start the game.
3. Create the player character quickly.
4. Wait on the TV screen.
5. Press A exactly when the final timer stage ends.

The input moment is the instant the countdown ends, which is when the fourth cue beep starts.

## Calibration

After entering the game, check the **TID** you actually received.

In the app:

1. Enter the actual TID in **Actual ID Check**.
2. Tap **Reverse Hit Delay**.
3. If there is no result, increase **Delay Window**.
4. Tap the reverse-search result.

The app fills the timer's **Delay Hit** field.

If the result is correct, tap **Apply Calibration** to update the timer.

## Retry

Repeat the same process after calibration.

If the hit Delay is only off by **±2**, you can try a few more times before updating calibration.

If the same Delay appears consistently, apply calibration.

By repeating this loop, you can stabilize the target Delay and obtain the desired TID/SID.
