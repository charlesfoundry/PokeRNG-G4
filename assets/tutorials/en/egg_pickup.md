# HGSS Egg Pickup

## Core Idea

Egg pickup decides:

- IVs
- inheritance

It depends directly on the main RNG Advance.

This assumes egg generation RNG is already complete: the target egg has already been generated, has not been received, and the player is saved in front of the Day-Care Man.

## Initial Advance

The HGSS Day-Care route has several NPCs that can advance frames, so estimate the initial Advance before pickup.

If two roamers are active:

- initial Advance is usually treated as 2
- rarely it can be 3
- start searches with 2 first

## Search Stage

Enter:

- parent IVs
- year
- Delay range
- pickup Advance range

Then search and choose the desired result.

## Receiving the Egg

Because the save is already in front of the Day-Care Man, the goal is to receive the egg as fast as possible.

Flow:

1. Follow the timer to set the DS clock.
2. Start the game.
3. Enter the save.
4. After loading, immediately mash A.
5. Receive the egg as fast as possible.

This usually picks up an egg near the initial Advance, such as Advance 2.

## Seed Confirmation

After receiving the egg, open the phone and verify which seed was hit.

At that point Advance may already be around 10 or even 20, so make enough calls to identify the hit seed accurately.

After every call, tap the exact matching E/K/P result in the app in the same order.

After confirming the hit seed, manually tap that seed result so the app records the seed you hit.

If the seed is confirmed, the target IV result was usually hit as well.

If the seed was missed:

1. Tap the seed result you actually hit.
2. The app fills the Egg Delay actually hit in this attempt.
3. The egg timer is updated only after you tap Apply Calibration.
4. Reset the game.
5. Repeat the pickup attempt.

If the hit delay differs from the target delay by only **±2**, you can avoid updating calibration yet and try a few more times with the current calibration first.

If the same offset appears consistently, then apply the calibration.
