# HGSSタマゴ生成

## 基本方針

タマゴ生成では **Egg Frame** を使います。

生成段階では次が決まります。

- PID
- 性格
- 性別
- 特性
- 色違い

最も簡単な目標は、検索時に **Egg Frame 1** の結果を選ぶことです。seed を命中できれば、ゲーム内で歩いて育て屋じいさんに最初のタマゴを生成させます。

## Egg Frame を進める操作

Egg Frame を進める操作：

- たんパンこぞうゴロウへの電話
- タマゴを断る
- 類似するタマゴ生成関連操作

Egg Frame を進めない操作：

- ウツギ博士への電話
- ジャグラーへの電話

ウツギ博士やジャグラーの電話は seed 確認に使えますが、Egg Frame 消費として自動扱いしません。

## 検索段階

1. Egg Frame 範囲を設定します。
2. 目標 PID を選びます。
3. タマゴ用タイマーへ送ります。
4. 乱数を開始します。

最初の推奨設定：

- 最小 Egg Frame：1
- 最大 Egg Frame：1

## seed を命中する

1. タイマーに従ってゲームに入ります。
2. すぐ電話を開きます。
3. ウツギ博士またはジャグラーに電話して seed 命中を確認します。

電話を 1 回かけるたびに、app で完全に一致する E/K/P を同じ順番でタップします。

命中 seed を確認した後も、その seed 結果を手動でタップして、app に今回命中した seed を記録させます。

外した場合：

1. 実際に命中した seed 結果をタップします。
2. app は今回実際に命中したタマゴ delay を自動入力します。
3. タマゴ用タイマーが更新されるのは、ユーザーが校準を適用した後です。
4. 再挑戦します。

命中 delay と目標 delay の差が **±2** 程度なら、すぐ校準を更新せず、現在の校準のまま数回試してもかまいません。

同じずれが安定して出る場合に、校準を適用します。

命中した場合：

1. 電話を閉じます。
2. 歩き始めます。
3. 育て屋じいさんがタマゴを生成するまで待ちます。

## 目標タマゴを固定する

タマゴが生成された後：

1. 育て屋じいさんの前でレポートします。
2. タマゴを受け取って孵化します。
3. 性格、色違い、性別が目標通りか確認します。

目標通りなら：

1. 孵化後の状態は保存しません。
2. ゲームをリセットします。
3. 育て屋じいさんの前のレポートに戻ります。

この時点の想定状態は、目標タマゴが生成済み、未受取で、育て屋じいさんの前にレポート済みです。次に受取個体を乱数します。
