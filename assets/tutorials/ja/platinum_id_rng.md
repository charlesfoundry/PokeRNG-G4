# Pt ID乱数

## 目的

Pt の ID 乱数は、新規セーブで狙った **TID/SID** を出すための手順です。

その TID/SID が優秀な PID と合えば、後でその PID を色違いにできます。

## SID範囲を選ぶ

1. **ID乱数** ページを開きます。
2. **SID検索** を押します。
3. 狙いたい **TID** を入力します。
4. **最小SID** と **最大SID** は空欄でも構いません。
5. **優秀SID検索** を押します。

優秀 SID 検索ページへ移動します。

## 優秀SIDを探す

優秀 SID ページでは、各 TSV で色違いにできる優秀 PID グループを確認できます。

おすすめ設定：

- 並び順は **性格最多**。
- **最低個体値** は、優秀個体として判定するための最低個体値です。

**優秀SIDを検索** を押して、結果カードを確認します。

各カードには、その TSV で狙える優秀 PID が表示されます。

気に入ったカードをタップします。

app は前のページへ戻り、対応する SID 範囲を自動で入力します。

## 実際に到達できるSIDを検索する

SID検索ページに戻ったら：

1. **年** を入力します。
2. **最小Delay** と **最大Delay** を設定します。
3. **SID検索** を押します。

ここで検索するのは、選んだ年と Delay 範囲で実際に命中できる SID です。

前の手順で出した理論上の優秀 SID 全体ではなく、実機の時間で狙える結果だけを絞り込みます。

おすすめ：

- 年は、DS 上でゲームを開始したい年です。
- 最小Delayは **5000** 以上を推奨します。
- 新規ゲームでは主人公作成に時間がかかるため、低すぎる Delay は現実的ではありません。
- 検索が重い場合は、最小Delayと最大Delayの差を小さくしてください。

## 時刻を選ぶ

狙いたい SID 結果をタップすると、ID乱数ページへ戻ります。

次に：

1. **Seed時間を取得** を押します。
2. 月と日を入力します。
3. 使いたい時刻を選びます。

ここで入力した月日が、トレーナーカードに表示されるゲーム開始日になります。

時刻をタップすると app が戻り、目標 Delay がタイマーに入ります。

## 実機操作

タイマーに従って操作します。

1. DS の時刻を設定します。
2. ゲームを起動します。
3. 主人公を素早く作成します。
4. テレビ画面で待機します。
5. 最後のタイマーが終わる瞬間に A を押します。

入力タイミングは、カウントダウン終了の瞬間です。4 回目の音が鳴り始める瞬間に押します。

## 校準

ゲームに入ったら、実際に出た **TID** を確認します。

app で：

1. **実際ID確認** に実際の TID を入力します。
2. **命中Delayを逆算** を押します。
3. 結果がない場合は **Delay範囲** を広げます。
4. 逆算結果をタップします。

app はタイマーの **命中Delay** に結果を入力します。

結果が正しければ、**調整を反映** を押してタイマーを更新します。

## 再試行

校準後、同じ手順を繰り返します。

命中 Delay の差が **±2** 程度なら、すぐに校準を更新せず、何回か試しても構いません。

同じ Delay が安定して出るなら、校準を反映します。

この流れを繰り返すことで、目標 Delay を安定して命中し、狙った TID/SID を取得できます。
